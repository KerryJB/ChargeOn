# ChargeOn
Windows laptop battery conditioner (GUI version)

## Components
* Arduino Nano board with 433MHz transmitter module and (optional) 433MHz receiver module
* 433MHz remote outlet switch (tested with Etekcity ZAP and Dewenwils)
